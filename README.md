# 🖥️ CSV Cleaner UI

Interface web para interação com o sistema de limpeza de arquivos CSV.

---

## 🚀 Sobre o projeto

O **CSV Cleaner UI** oferece uma interface amigável para upload e processamento de arquivos CSV, facilitando o uso do sistema por usuários não técnicos.

---

## ⚙️ Funcionalidades

* 📂 Upload de arquivos CSV
* 📊 Visualização de dados
* ⚡ Integração com API de processamento
* 🎨 Interface moderna

---

## 🛠️ Tecnologias

* TypeScript
* Frontend moderno (React ou similar)
* Integração com APIs REST

---

## ▶️ Como rodar

```bash
git clone https://github.com/fernandavsp/csv-cleaner-ui.git
cd csv-cleaner-ui
npm install
npm run dev
```
